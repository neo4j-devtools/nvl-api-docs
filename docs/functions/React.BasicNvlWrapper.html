Function BasicNvlWrapper

  • BasicNvlWrapper(props: Omit<BasicReactWrapperProps & HTMLProps<HTMLDivElement>, "ref"> & RefAttributes<IncludeMethods<NVL>>): ReactNode

  • A basic React wrapper for the NVL class. It takes the BasicReactWrapperProps as properties which are passed to the NVL constructor. Any changes in properties will be reflected in the NVL instance by calling the corresponding methods.

    Example

    <BasicNvlWrapper
     nodes={nodes}
     rels={relationships}
     nvlOptions={options}
     nvlCallbacks={callbacks}
    />

    would be equivalent to

    const myNvl = new NVL(container, nodes, relationships, options, callbacks)

    Example

    When nodes and/or relationships are updated in the React wrapper, the NVL instance will be updated accordingly:

    const NvlComponent = () => {
      const [nodes, setNodes] = useState<Node[]>([{ id: '0' }, { id: '1' }])

      const addElements = () => {
      const newNodes = [...nodes, { id: nodes.length.toString() }]
      setNodes(newNodes)
    }

    return <div>
      <BasicNvlWrapper nodes={nodes} rels={[]} />
      <button onClick={addElements}>Add Graph Elements</button>
    </div>
    }

    would be equivalent to

    const container = document.createElement('div')
    const myButton = document.createElement('button')
    const nodes = [{ id: '0' }, { id: '1' }]
    const myNvl = new NVL(container, nodes, [])
    myButton.addEventListener('click', () => {
      const newNodes = [...nodes, { id: nodes.length.toString() }]
      myNvl.addAndUpdateElementsInGraph(newNodes, [])
    })

    Example

    If you want to access the NVL class outside of the React wrapper you can use a reference of NVL to call its methods:

    const NvlComponent = () => {
      const nvlRef = useRef<NVL>()

      return <div>
        <BasicNvlWrapper
          nodes={[{ id: '0' }, { id: '1' }]}
          rels={[{ from: '0', to: '1', id: '10' }]}
          ref={nvlRef}
        />
        <button onClick={() => nvlRef.current?.fit([0, 1])}>Zoom to Nodes</button>
      </div>
    }

    Example

    If you want to pass events to the NVL instance you can do so by passing them as properties to the React wrapper:

    <BasicNvlWrapper
     nodes={nodes}
     rels={relationships}
     onClick={(event) => console.log(event)}
    />

    would be equivalent to

    const container = document.createElement('div')
    const nodes = [{ id: '0' }, { id: '1' }]
    const myNvl = new NVL(container, nodes, [])
    container.addEventListener('click', (event) => console.log(event))

    For more about interactivity, check out the NVL.hittest method, the Interaction Handlers module and the InteractiveNvlWrapper.
